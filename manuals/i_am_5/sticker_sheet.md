# Sticker Sheet & Coloring Pages

Print these pages on sticker paper or regular paper!

---

## Robot Alphabet Stickers

```
A = Antenna    B = Battery    C = Camera    D = Drive
E = Eye        F = Fan        G = Gear      H = Handle
I = I2C        J = Joystick   K = Key       L = Laser
M = Motor      N = Nut        O = Obstacle  P = Pi (computer)
Q = Quiet      R = Robot      S = Sensor    T = ToF
U = USB        V = Voltage    W = Wheel     X = X-axis
Y = Yaw        Z = Zoom
```

Cut out each letter and stick them on your robot!

## Coloring Page 3: Advika's Room Map

Draw your room here. Mark where Advika should go!

```
+----------------------------------+
|                                  |
|    [COUCH]                       |
|                                  |
|              [TABLE]             |
|                                  |
|    [BED]              [DOOR]     |
|                                  |
|         [START]                  |
|                                  |
+----------------------------------+
```

- Draw a RED path for Advika to follow
- Draw a BLUE X where she should stop
- Draw a GREEN circle for a safe zone
