# "I am 5" -- Child-Friendly Robot Manual

This manual is designed for children ages 5 and up who want to interact with, learn about, and befriend Advika 3.0.

## Manual Structure

| File | Purpose | Reading Level |
|------|---------|---------------|
| `cover.md` | Title page and introduction | Parent reads aloud |
| `meet_advika.md` | Getting to know your robot | Ages 5+ with help |
| `safety_rules.md` | Important safety guidelines | Ages 5+ with parent |
| `fun_commands.md` | Commands and games | Ages 6+ |
| `sticker_sheet.md` | Printable activities | All ages |

## How to Use This Manual

1. **Start with the cover** -- Read the introduction together
2. **Learn the safety rules** -- Go through Chapter 2 carefully with a parent
3. **Try simple commands** -- Start with basic driving commands
4. **Print the sticker sheet** -- Let kids customize their robot experience
5. **Progress to challenges** -- Work through the challenge levels together

## Parent/Guardian Notes

- This manual assumes a responsible adult is present during all robot interactions
- The safety rules in Chapter 2 should be reviewed before EVERY play session
- Encourage children to ask questions and explore, but within safe boundaries
- Use this as an opportunity to teach basic robotics, programming concepts, and safety awareness

## Printing Recommendations

- Print sticker sheets on **Avery 8165** or equivalent full-sheet label paper
- Coloring pages work best on standard 8.5x11" paper
- Consider laminating the safety rules page for durability

## Contributing

If your child creates new commands, drawings, or robot adventures, consider sharing them with the Advika community!
